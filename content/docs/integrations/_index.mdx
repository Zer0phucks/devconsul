# Platform Integrations

Connect Full Self Publishing to blogs, social media, and email platforms to automate your content distribution.

## Available Integrations

### Blog Platforms

- **[WordPress](/docs/integrations/wordpress)** - Self-hosted & WordPress.com blogs
- **[Ghost](/docs/integrations/ghost)** - Modern publishing platform
- **[Medium](/docs/integrations/medium)** - Built-in audience network

### Social Media

- **[Twitter/X](/docs/integrations/twitter)** - Tweets and threads
- **[LinkedIn](/docs/integrations/linkedin)** - Professional updates
- **[Facebook](/docs/integrations/facebook)** - Pages and groups
- **[Reddit](/docs/integrations/reddit)** - Community posts

### Email Platforms

- **[Resend](/docs/integrations/resend)** - Modern email API
- **[SendGrid](/docs/integrations/sendgrid)** - Enterprise email delivery
- **[Mailchimp](/docs/integrations/mailchimp)** - Marketing automation

### Custom

- **[Webhooks](/docs/integrations/webhooks)** - Custom integrations

## Quick Start

1. Navigate to **Settings > Platforms**
2. Choose your platform
3. Follow platform-specific setup guide
4. Verify connection status
5. Configure publishing settings

## Authentication Methods

**OAuth 2.0** (Recommended)
- WordPress.com, Medium, LinkedIn, Facebook, Twitter, Mailchimp
- One-click authorization
- Automatic token refresh

**API Keys**
- Ghost, Resend, SendGrid, Reddit
- Manual credential entry
- Secure encrypted storage

## Multi-Platform Publishing

You can connect multiple platforms simultaneously:

- **Same Content, Multiple Platforms:** Adapt one piece for all channels
- **Platform-Specific Content:** Generate different content per platform
- **Custom Publishing Rules:** Define which platforms get which content types
